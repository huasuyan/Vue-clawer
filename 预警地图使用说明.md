# 预警地图实现说明

## 已完成的功能

### 1. 基础地图
- 使用高德地图 API 2.0
- 地图中心定位在厦门市（118.1, 24.48）
- 支持缩放、拖拽等基本操作

### 2. 区域边界
- 蓝色虚线绘制监控区域边界
- 使用 AMap.Polygon 实现

### 3. 统计面板（左侧）
- 预警人员统计
- 预警事件统计
- 预警地点统计
- 预警车辆统计
- 点击可筛选对应类型

### 4. 标记点
- 不同类型使用不同图标（SVG）
- 不同级别使用不同颜色
  - 一级预警：红色 #FF4444
  - 二级预警：橙色 #FF8800
  - 三级预警：黄色 #FFCC00
- 点击标记显示详细信息

### 5. 控制按钮（右上角）
- 预警人员筛选
- 预警事件筛选
- 预警地点筛选

### 6. 图例说明（底部）
- 显示各类标记的含义

## 使用步骤

### 1. 申请高德地图 Key
访问 https://console.amap.com/dev/key/app
- 注册/登录高德开放平台
- 创建应用
- 添加 Key（选择 Web 端 JS API）
- 复制 Key

### 2. 配置 Key
在 `src/views/document/WarningMap.vue` 第 77 行：
```javascript
key: '你的高德地图key', // 替换为你申请的 Key
```

### 3. 访问页面
```
http://localhost:5173/document/warning-map
```

## 自定义数据

修改 `warningData` 数组添加你的预警数据：

```javascript
const warningData = ref([
  {
    type: 'event',      // 类型: person/event/location/vehicle
    level: 1,           // 级别: 1/2/3
    position: [118.1, 24.48], // 经纬度
    name: '预警事件1'   // 名称
  }
])
```

## 接入后端 API

在 `initMap` 函数中添加数据请求：

```javascript
const initMap = async () => {
  // ... 地图初始化代码
  
  // 获取预警数据
  const response = await fetch('/api/warning/list')
  const data = await response.json()
  warningData.value = data
  
  addMarkers(AMap)
}
```

## 主要 API 说明

### AMap.Marker - 标记点
```javascript
new AMap.Marker({
  position: [lng, lat],  // 位置
  icon: 'icon_url',      // 图标
  title: 'title'         // 标题
})
```

### AMap.Polygon - 多边形
```javascript
new AMap.Polygon({
  path: [[lng, lat], ...],  // 路径
  strokeColor: '#4A90E2',   // 边框颜色
  strokeStyle: 'dashed',    // 边框样式
  fillColor: 'transparent'  // 填充颜色
})
```

### AMap.InfoWindow - 信息窗口
```javascript
new AMap.InfoWindow({
  content: '<div>内容</div>',
  offset: new AMap.Pixel(0, -30)
})
```
